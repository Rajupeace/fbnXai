# FRIENDLY NOTEBOOK SYSTEM

## Overview
Friendly Notebook (formerly FBN XAI) is an advanced educational management system designed for high-fidelity academic tracking, real-time communication, and detailed analytics.

## Created By
**Bobby Martin**

## Features
- **Student Dashboard**: Real-time tracking of attendance, marks, and schedules.
- **Faculty Dashboard**: Management of classes, materials, examinations, and attendance.
- **Admin Sentinel**: Governance and system-wide monitoring.
- **Neural Core (VuAiAgent)**: AI-driven assistance for students and faculty.
- **Chronos Protocol**: Advanced scheduling and timeline management.
- **Nexus Forge**: Laboratory schedule and resource management.

## Tech Stack
- **Frontend**: React.js, Framer Motion, Vanilla CSS (Nexus Design System)
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **AI**: Integrated Neural Core

## Setup
1. `npm install`
2. `npm start`
